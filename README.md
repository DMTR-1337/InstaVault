# Capture the Flag: InstaVault

### **Description**
"InstaVault" is a modernized image hosting service built on an aging foundation. While the UI looks fresh, the backend is riddled with "convenient" legacy features left behind by a lazy administrator.

**Your Mission:** Get the flag from /root/flag.txt

---

### **Getting Started**
1. Ensure **Docker** and **Docker Compose** are installed.
2. Start the environment:
   ```bash
   https://github.com/DMTR-1337/InstaVault.git
   cd InstaVault-main
   docker-compose up -d
   ```
3. Capture the Flag!
